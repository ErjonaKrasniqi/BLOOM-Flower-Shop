Flower Shop Website

This project is a simple "Flower Shop website" built using HTML and CSS.  
It is designed to showcase the beauty and elegance of floral bouquets while providing a pleasant browsing experience for users.


Features
Clean and modern design  
Section for best-selling bouquets  
"About Us" and "Contact" pages  
Fully responsive layout for mobile and desktop  
Soft, nature-inspired color palette  

Technologies Used
HTML5 for website structure  
CSS3 for styling and basic animations
